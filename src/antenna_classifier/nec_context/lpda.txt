Log-Periodic Dipole Array (LPDA) — NEC Modeling Reference

Structure: Array of dipole elements of decreasing length, connected by a
crossed transmission line. Elements are spaced logarithmically. Provides
broadband coverage across a wide frequency range.

Key rules:
- Each element is one GW card (horizontal dipole), all at the same Z height.
- Elements extend along Z (or Y), spaced along X with decreasing length.
- Adjacent elements are connected by TL cards with alternating phase (negative Z0).
- Feed the shortest (front) element.
- Negative Z0 in TL cards means the transmission line has a 180° phase reversal
  (cross-connected feed line), which is essential for LPDA operation.
- Use GS to scale if dimensions are in wavelengths.
